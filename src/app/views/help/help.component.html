Help

This section describes PED website use. Information about deposition in PED can be found on the Deposition page. For information about RESTful endpoints and the output format see the API documentation below.

Graphical User Interface

The PED graphical user interface allows easy access to PED depositions in a graphical way. The data can be searched from the Browse page or from the search box in the Home page.

Browse page

The Browse page is composed of a result table, a sidebar (left) and a search bar (above the table).

  • The table lists search results. By default it shows the first 20 entries of the database sorted alphabetically by the PED identifier (second column). You can change the table sorting by clicking on column headers.
  • The sidebar allows users to select columns to visualize and customize the results of their search.
  • The search box right above the table offers a free text search against all entry values. Alternatively, it allows users to specify the field to search against, i.e. entry identifier, UniProt ACC, term. It is possible to combine multiple criteria and create complex queries with an intersection ("AND" operator).

Protein-centric page

Users can look for a specific protein in PED by selecting its corresponding UniProt identifier, e.g. P38634 for protein Sic1 from Saccharomyces cerevisiae. The protein-centric page features three different sections, described here from the top to the bottom:

  1. a top panel with general protein information, including the organism, function and cross-references;
  2. a graphical view of the listed ensembles, including the identifier of the ensemble page, e.g. PED00001, a detailed description of the ensemble and the experimental tags associated.

Users can filter the ensembles using the search box in the center of the protein-centric page, and perform a free text search against the deposited ensembles associated with the protein.

It is also possible to download all the ensembles associated with a protein by clicking on the “Download all” button in the center of the protein-centric page; alternatively users can select single ensembles and download them by clicking on the “Download selected” button.

Ensemble-centric page

Users can look for a specific experimental entry in PED by selecting its corresponding PED identifier, e.g. PED00001.

The ensemble-centric page features three different sections, described here from the top to the bottom:

  1. the PED identifier and a description of the ensemble, e.g. “PED00001 - Structural ensemble of pSic1 (1-90) with phosphorylations at Thr5, Thr33, Thr45, Ser69, Ser76, Ser80”, followed by a top panel with general ensemble information, including the authors of the ensemble, the expression organism, a description of the methods and the associated experimental tags, and the literature reference, where available;
  2. the feature viewer, a graphical view of the ensemble, its chains and sequences, and information information related to:
    1. post-translational modifications (PTMs) and mutations characterizing the protein,
    2. the secondary structure entropy of a given position reflecting the variation of helix, beta, coil secondary structure classification,
    3. the relative ASA along the chain.
  3. (i.) the structure viewer, where users can visualize and switch between the structures of the 10 most representative conformers of the ensemble, based on the Rg value, (ii.) the Ramachandran plot and (iii.) an Rg boxplot.
